# 👥 HR Admin Mini 프로젝트 기반 React 스터디

## 🎯 목표

React 입문자를 위한 실전형 스터디입니다.  
실무에서 자주 쓰이는 HR Admin 기능을 직접 구현하며,  
React의 기본 개념과 상태 관리 흐름을 체득하는 것이 목표입니다.

## 🗓 스터디 일정 (총 6주)

| 주차 | 주제 |
|------|------|
| 1주차 | TeamMemberCard 만들기 |

| 6주차 | 데모 발표 + 회식 (오프라인) |
